# About Cozy Hollow

The Food Delivery System follows a cottagecore-inspired design, creating a warm and comforting experience for users. Inspired by the feeling of small woodland homes and cozy tree hollows, the application aims to make food ordering feel personal, peaceful, and inviting.

The concept originated from the idea that little critters feel safe and cozy inside a tree’s hollow — the same warmth and comfort our food brings to customers. Through soft visuals, earthy tones, and a welcoming interface, the app delivers not only meals but also a sense of home and comfort.

# 👥 Customer Features

  🍔 Browse food menu and categories 
  
  🛒 Add to cart and manage orders
  
  📦 Place and track orders in real-time
  
  🧾 View order history
  
  ⭐ Responsive and user-friendly interface
  
# 🛠️ Admin Features

  📋 Manage food items and categories
  
  📦 Monitor customer orders
  
  👥 Manage users and riders
  
  📊 Dashboard analytics and order overview
  
# 🛵 Rider Features

  📍 View assigned deliveries
  
  🟢 Edit rider status (Online / Offline)
  
  ✅ Accept and complete delivery orders
  
  🚚 Real-time delivery status updates
  
  📱 User-friendly rider dashboard
